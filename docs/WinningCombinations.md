Title: WINNING_COMBINATIONS

Source: [src/winning-combinations.js](src/winning-combinations.js)

Description:

- Exports `WINNING_COMBINATIONS`, an array of 8 winning triplets for tic-tac-toe.

Structure:

- Each combination is an array of three objects `{ row, column }` describing board coordinates.

Usage:

- Used by `App` to determine whether a player has three matching symbols in a winning pattern.
